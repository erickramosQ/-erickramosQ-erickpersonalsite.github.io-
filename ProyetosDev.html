PCS Mascotas

Tiene por objetivo interactuar con un sistema de control, en este caso un dispensador de comida electronico construido por mi persona, es una aplicacion para control IoT.


La aplicación interactua con el sistema de control mediante el servicio de firebase (Realtime Database) para realizar acciones mientras uno no esta en casa, de esta manera se puede alimentar a la mascota mientras uno trabaja o estudia. La programación por parte del microcontrolador se realizó de manera aislada de la aplicación, Firebase actua como intermediario.


Web Intranet entidad bancaria

Este es un sistema web que lista registros e imagenes tipo TIFF hoja por hoja que interactua con un servicio windows y hace un hover para ver la hoja como zoom, además de esto se puede clasificar cada una de estos registros y modificar los datos de los clientes, en caso de que exista algun error en los datos traidos por el servicio windows, la Web se realizo con HTML5, Javascript y Materialize que es framework CSS responsive basado en Material Design.

Web de Reportes Cañeros Guabira

Este es un sistema web que consume un WebService que realicé con Node.js, este servicio se encuentra en el servidor del cliente donde se encuentra su base de datos, en tema de frontend la Web se realizó con HTML5, Javascript y Materialize que es framework CSS responsive basado en Material Design, lo que hace la web es enviar al servicio tanto para el login como para las consultas los datos de entrada y con la respuesta genera un reporte en pdf o excel, esto para fines del cliente.
